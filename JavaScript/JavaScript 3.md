#### 짧은 조건문(short-circuit)

A || B ⇐ A 또는 B, 둘 중 하나가 참이면 참 = A와 B, 둘 모두 거짓이어야 거짓		  (논리OR)

A && B ⇐ A와 B, 둘 다 모두 참이어야 참 = A 또는 B 둘 중 하나가 거짓이면 거짓	(논리AND)
